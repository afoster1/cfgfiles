Far Manager Configuration
-------------------------

* Load a saved configuration file: Far.exe -import [filename]
* Save the current configuration to a file: Far.exe -export [filename]

Macros
------
* Start recording a new macro: ctrl+.
* Stop recording a macro: ctrl+.
* Save macros: lm:save
* Load macros: lm:load
* Show file context menu (text): rclk_txt:
* Show file context menu (gui): rclk_gui:

Using Provided Macros
---------------------
* Copy macros from "Far\Profile\Macros\internal" into "%FARPROFILE%\Macros\internal"
* Load the macros (or restart Far): lm:load
